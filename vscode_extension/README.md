# SBARGOLD VS Code Extension

Questa estensione fornisce supporto per il linguaggio **SBARGOLD** in Visual Studio Code.

## Funzionalità
- **Syntax Highlighting**: Evidenziazione della sintassi per comandi, stringhe, numeri e commenti.
- **Snippet**: (Pianificati)
- **Bracket Matching**: Corrispondenza parentesi per blocchi `SBARGOLD{ ... }`.

## Installazione Manuale
1. Copia la cartella `vscode_extension` nella tua cartella delle estensioni di VS Code:
   - Windows: `%USERPROFILE%\.vscode\extensions\sbargold`
   - Mac/Linux: `~/.vscode/extensions/sbargold`
2. Riavvia VS Code.
3. Apri un file `.sbg` e goditi l'evidenziazione!

## Sviluppo
Per testare l'estensione durante lo sviluppo, apri la cartella `vscode_extension` in VS Code e premi `F5` per avviare una finestra di debug con l'estensione caricata.
