## 📝 Descrizione
Implementato il supporto per Input/Output su file.
Ora SBARGOLD può interagire con il filesystem, permettendo la persistenza dei dati.

## 🛠️ Nuove Funzionalità
- **Scrittura File (`SBARGOLD>>`)**:
    - Sintassi: `SBARGOLD>> "filename.txt" "contenuto"`
    - Scrive (o sovrascrive) il contenuto nel file specificato.
- **Lettura File (`SBARGOLD<<`)**:
    - Sintassi: `contenuto SBARGOLD<< "filename.txt"`
    - Legge l'intero contenuto di un file in una variabile stringa.
